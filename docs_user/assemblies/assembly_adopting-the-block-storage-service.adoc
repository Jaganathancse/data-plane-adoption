= Adopting the {block_storage}

To adopt a {OpenStackPreviousInstaller}-deployed {block_storage_first_ref} into {rhos_long}, perform the following actions:

* Check existing limitations.
* Consider the placement of the Block Storage services.
* Prepare the {OpenShift} nodes where the volume and backup services run.
* Create the manifest based on the existing `cinder.conf` file.
* Deploy the {block_storage}.
* Validate the new deployment.
